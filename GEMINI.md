# Project Overview

This project is a personal website created for the "Website Development" competition (IT Stars 2026). The theme of the site is "My Hobby: a source of inspiration and personal growth," and it showcases the author's passion for IT, Linux, and DevSecOps.

The website is a static, multi-page site built with **HTML, CSS, and JavaScript**, utilizing the **Bootstrap 5** framework for styling and layout. The aesthetic is a dark, terminal-inspired theme.

## Key Files

*   `index.html`: The main landing page.
*   `css/style.css`: Custom styles to create the terminal-like theme.
*   `js/script.js`: Contains a simple interactive quiz.
*   `README.md`: Provides context about the competition.
*   `Критерии конкурса сайт.docx.md`: Contains the detailed rules and evaluation criteria for the competition.

## Building and Running

This is a static website and does not require a build process.

*   **To run the project:** Simply open the `index.html` file in a web browser.
*   **To run tests:** There are no automated tests for this project.

## Development Conventions

*   **Structure:**
    *   HTML files are in the root directory.
    *   CSS files are in the `css/` directory.
    *   JavaScript files are in the `js/` directory.
*   **Styling:** The project uses Bootstrap 5 for its base styles and responsiveness, with custom overrides in `css/style.css` to achieve the desired look and feel.
*   **Interactivity:** JavaScript is used for small interactive elements, like the prompt in `js/script.js`.
*   **AI Usage:** The project notes in the footer that AI assistants were used to help with structure and code optimization.
