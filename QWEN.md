# Проект — Статический сайт‑конкурс «Разработка сайта»

## Описание проекта
Это небольшая статическая веб‑страница, созданная для участия в конкурсе «Разработка сайта». Тема сайта — **«Моё хобби: источник вдохновения и личностного роста»**. Главная цель — за короткое время (2 ч) продемонстрировать, как выбранное хобби влияет на развитие личности, помогает генерировать новые идеи, ставить цели и находить возможности.

## Как использовать проект

1. **Открыть в браузере**  
   Достаточно открыть `index.html` в любом современном браузере (двойной клик или `file://`‑путь).

2. **Запуск локального сервера (по желанию)**  
   Если нужен live‑reload или CORS‑поддержка, можно запустить простой HTTP‑сервер:

   ```bash
   # Python 3
   python -m http.server 8000 --directory /home/lorriant/work/istar2026WebP
   # Затем открыть http://localhost:8000 в браузере
   ```

3. **Настройка стилей**  
   - Все стили находятся в `css/style.css`.  
   - При необходимости изменить цветовую схему, замените переменные `#198754` на желаемый HEX‑код.  
   - Шрифты задаются в `body { font-family: ... }`.

4. **Добавление поведения**  
   - Поместите JavaScript‑код в `js/script.js`.  
   - Подключите новые библиотеки, если потребуется (например, jQuery, Vue.js), добавив соответствующие `<script>`‑теги в `index.html`.

5. **Тестирование и проверка**  
   - Убедитесь, что страница корректно отображается на разных устройствах.  
   - Проверьте соответствие **Критериям конкурса** (см. `Критерии конкурса сайт.docx.md`).

## Конвенции разработки

- **HTML‑структура**: семантическая, использует Bootstrap‑подобные классы (`nav-link`, `card`, `btn-outline-success`).
- **CSS**: оформлен в одном файле `css/style.css`, использует чистый CSS без препроцессоров. Все анимации реализованы через `transition` и `transform`.
- **JS**: пока отсутствует, но проект подразумевает модульную структуру (отдельный каталог `js/`).
- **Git**: проект находится под контролем версии; изменения коммитятся с описательными сообщениями, соответствующими требованиям конкурса.
